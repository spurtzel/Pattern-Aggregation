# On-Demand Pattern Aggregation in Event Networks

## Requirements

### Build Tools
- Make
- CMake (version >= 3.12.0)
- C++ compiler supporting C++20 or higher


### Python Script Requirements
- Python3 (version >= 3.8)
- Required Python Libraries:
    - argparse
    - json
    - matplotlib
    - numpy
    - random
    - scipy
    - time
